JR Elite Builders PHP template (v001)

What you get:
- index.php, about.php, services.php, contact.php
- includes/header.php and includes/footer.php
- Uses the same theme asset paths referenced on jrelitebuilders.com (assets/css/*, assets/js/*, assets/img/*)

How to run:
1) Put these PHP files into your web root.
2) Ensure you have the theme "assets/" folder in the same directory:
   - assets/css/
   - assets/js/
   - assets/img/
3) Visit /index.php

Notes:
- contact.php includes an OPTIONAL mail() handler. If mail() is disabled on your host, replace it with SMTP (PHPMailer) or your form provider.
- gallery.php is referenced in menus but not included here (you can add it next).
